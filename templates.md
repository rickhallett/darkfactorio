# darkfactorio templates

Use these templates to capture outputs from slash agents with a stable file layout.

- `specs/spec-template.md`
- `scenarios/scenario-suite-template.md`
- `org/org-redesign-template.md`
- `migration/legacy-migration-roadmap-template.md`
- `talent/talent-strategy-template.md`
- `playbooks/level4-pilot-checklist.md`

Suggested naming:

- Specs: `specs/YYYY-MM-DD-<initiative>.md`
- Scenario suites: `scenarios/YYYY-MM-DD-<initiative>-suite.md`
- Org redesign: `org/YYYY-MM-DD-org-redesign.md`
- Migration plan: `migration/YYYY-MM-DD-legacy-roadmap.md`
- Talent strategy: `talent/YYYY-MM-DD-talent-plan.md`
